编译C语言时使quickfix快速修复错误
=========================

对于一个Makefile管理的C语言或者C++语言工程。我们可以使用下面的Ex命令来编译工程。
```viml
:make
```

当出现多个编译错误时，可以使用下面的命令来跳到下一个错误：
```viml
:cnext
```

如果需要查看详细的报错信息，使用下面的命令：
```viml
:cc
```

其他的修复命令
--------------
* `:clist` : 查看所有的错误列表
* `:clist!` : 查看所有的错误列表，同时显示错误提示的上下文
* `:cfirst` : 跳到第一个错误
* `:clast` : 跳到最后一个错误，与`:cfirst`相对
* `:cprevious` : 跳到前一个错误，与`:cnext`相对
